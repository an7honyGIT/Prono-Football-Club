Calcul des points sur un match

  • Vous marquez 3 points si la bonne équipe remporte le match.
  • Vous marquez 5 points si la bonne équipe remporte le match et vous avez le bon nombre de buts pour l'une des deux équipes.
  • Vous marquez 7 points si vous avez parié sur la bonne équipe et que vous avez la bonne différence de buts.
  • Vous marquez 10 points si vous avez parié sur le score exact.

Phase éliminatoire :

  • Le calcul des points sur le résultat du match est identique.
  • Vous marquez 5 points supplémentaires si la bonne équipe se qualifie.

Quitte ou Double :

Lorsque vous parier sur un match, vous avez la possiblité de jouer "quitte ou double". Si vous avez misé sur l'équipe qualifiée alors vous doublez vos points sur ce match. En revanche, une pénalité de 10 points vous sera attribuée si la seconde équipe se qualifie.

Exemple : vous misez "quitte ou double" sur le match France - Argentine.

  • Mise : 2-1 / Résultat: 2-1. Vous gagnez (10 + 5) x 2 = 30 points
  • Mise : 2-1 / Résultat: 1-1 et France qualifiée. Vous gagnez (0 + 5) x 2 = 10 points.
  • Mise : 2-1 / Résultat: Argentine qualifiée. Vous perdez 10 points.
